# Changelog

## 0.1.1

- Adds `Tiki Tiki Code Alert: Select Custom Sound`.
- Documents how to use a local licensed sound file.

## 0.1.0

- Initial VS Code extension.
- Plays a generated tiki-style WAV when active diagnostics remain broken while editing continues.
- Adds toggle, test sound, and silence commands.
- Adds configurable delay, cooldown, trigger mode, custom sound file, and status bar item.
